Scheme Interpreter

An interpreter for the Scheme language, powered by Python.

Functionality

The scheme interpreter:

  • Evaluates user expressions through syntactical and lexical analysis.
  • Outputs appropriate values after every command.
  • Catches all known exceptions and output useful error messages.
  • Lexically stores user-defined variables and procedures.
  • Calls functions via dynamic scoping via the mu keyword.
  • Implements most built-in expressions.
  • Implements recursive logic (there is no iteration in Scheme, only recursion).
